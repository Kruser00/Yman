# مبدل و ویرایشگر پیشرفته تصویر

یک ابزار قدرتمند و مدرن برای تبدیل و ویرایش تصاویر با رابط کاربری شیشه‌ای و پشتیبانی از فرمت‌های مختلف.

## ویژگی‌ها

- **پشتیبانی از فرمت‌های مختلف**: JPG, PNG, WEBP, AVIF, BMP, GIF, HEIC
- **ویرایش تصویر**: برش، تغییر اندازه، فیلترهای رنگی
- **پردازش دسته‌ای**: تبدیل چندین تصویر به صورت همزمان
- **مقایسه قبل و بعد**: اسلایدر تعاملی برای مقایسه تصاویر
- **امنیت 100%**: تمام پردازش‌ها به صورت لوکال در مرورگر انجام می‌شود
- **رابط کاربری مدرن**: طراحی شیشه‌ای با انیمیشن‌های روان

## اجرای محلی

**پیش‌نیازها**: Node.js

1. نصب وابستگی‌ها:
   ```bash
   npm install
   ```
2. اجرای برنامه:
   ```bash
   npm run dev
   ```

## ساختار پروژه

- `src/`: منبع اصلی برنامه
- `src/components/`: کامپوننت‌های رابط کاربری
- `src/lib/`: کتابخانه‌های پردازش تصویر
- `src/types/`: تعاریف TypeScript

## ساختار URL

برای دسترسی به بخش‌های مختلف، از URLهای زیر استفاده کنید:

- صفحه اصلی: `/`
- تبدیل فرمت خاص: `/convert/{فرمت-مبدا}-to/{فرمت-مقصد}`
  - مثال: `/convert/png-to-jpg`
  - مثال: `/convert/heic-to-png`

## ساختار پروژه

این پروژه با استفاده از React، TypeScript و Vite ساخته شده است.
